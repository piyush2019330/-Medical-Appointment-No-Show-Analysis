

# -Medical-Appointment-No-Show-Analysis
This project analyzes patient appointment data to understand the factors influencing no-shows. It includes data cleaning, exploratory data analysis (Python), SQL-based analysis, and an interactive Power BI dashboard to visualize key insights and patterns in patient attendance behavior.
